**Ganpati Cosmetics**
Ganpati Cosmetics is an e-commerce website designed to showcase and sell a wide variety of cosmetic products. The project leverages PHP and MySQL for server-side processing and database management, providing a user-friendly interface for customers to browse, purchase, and learn about cosmetic items.


**Features**
Product Catalog: Browse through a wide range of cosmetic products.
User Authentication: Secure login and registration for customers.
Product Search and Filter: Easily find specific products with search and filtering options.
Shopping Cart: Add products to the cart and review before purchase.
Order Management: Users can view their order history.
Admin Dashboard: For managing product inventory, orders, and customer accounts.

**Technologies Used**
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: PHP
Database: MySQL
Version Control: Git


**Usage**
Browse Products: Visitors can browse products without needing an account.
User Account: Customers must sign up and log in to add items to the cart and complete purchases.
Admin Functions: Accessible by administrators to manage inventory and customer orders.
